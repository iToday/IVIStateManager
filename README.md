# 设计初衷
车机中包含多种状态，在不同状态下，系统的功能表现不同；

如：

蓝牙电话时，需要打开显示屏和触摸屏，媒体静音，暂停播放，屏蔽部分功能按键等；

倒车时，需要打开显示屏，关闭触摸，媒体静音，暂停播放，屏蔽所有按键等；

待机时，需要打开显示屏，关闭触摸，屏蔽部分功能按键等；

需要设计一种机制统一管理状态的切换及行为，并且各状态下的行为可通过配置文件配置；
